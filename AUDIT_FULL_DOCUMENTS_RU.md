# Полный обзор всех документов репозитория IDT

Ниже перечислены **все** документы репозитория с кратким содержанием и выводами для аудита. Источники: извлечённый текст из PDF и DOCX (textutil).

## 1. Список документов и содержание

### 1.1 Теоретические и аксиоматические

| Документ | Формат | Содержание |
|----------|--------|------------|
| **Information-Dynamic Theory of Stable Actualization Regimes** | DOCX | Монография (~20 000 слов). Онтология: информационное поле, суперпозиция возможного, принцип самосогласованной устойчивости. Масса = структурная инерция; пространство, время, гравитация, квантовые явления выводятся из устойчивости. Аксиомы не перечислены здесь явно; математика: Z(ω), Φ = −ln Z, G (Fisher), κ = ∇ΦᵀG⁻¹∇Φ, **dω/dτ = −G⁻¹∇Φ**. Критерий фальсификации: det G = 0 при ∇Φ ≠ 0. Сравнение с Копенгагеном, Эвереттом, It from Bit, струнами. |
| **IDT Abstract EN** | DOCX | Краткий абстракт той же монографии: предмет, методология (онтология / модель / метафора), ключевые вопросы, научная новизна. Уровень — Foundations of Physics / Studies in History and Philosophy of Modern Physics. |
| **IDT Math Foundations Zenodo 0.0.1** | DOCX | Математические основания: аксиомы A0–A5, 16 канонически выведенных результатов (F01–F16), 8 расширений (CE). Включает Z, Φ, G, теорему Ляпунова (dΦ/dτ = −κ ≤ 0), устойчивость, структурную инерцию (CE01), лемму g_rr = C/r² для scale-family. Статусы: canon-derived / consistent-extension / conjecture. |
| **IDT Structural Inertia v1 4.1** | PDF/DOCX | **Определение:** m_IDT(u) = (uᵗGu)/(uᵗHu). **Теорема 1:** эта величина равна времени релаксации по направлению u в линеаризованной динамике d(δω)/dτ = −G⁻¹H δω. Примеры: G = I (евклидов случай), гауссово семейство с G ≠ I (меняется порядок и масштаб собственных значений). Связи с sloppy models, critical slowing down, репликаторными уравнениями, WFR. |
| **IDT Dimensional Inertia Law v1** | PDF/DOCX | **Закон размерной структурной инерции:** m*(d) = C r*² / [2A(4−d)] при Φ_eff = A/r² − B/r^(d−2). Только d = 3 даёт конечную положительную m*. Вывод из IDT-SI + IDT-D3. Численная проверка при A = B = C = 1. Открытые пробелы: GAP-E1, GAP-E2, GAP-E3, GAP-3 (τ → t). |
| **IDT D3 Final Revised 0 2** | DOCX | **IDT-D3:** устойчивые связанные режимы существуют единственным образом в **d = 3** для эффективного потенциала Φ_eff(r) = A/r² − B/r^(d−2). Теорема, усиленный вывод (конфайнмент из Fisher g_rr = C/r², связывание из лапласиана Δ_d 𝒜 = 0). Приложения: log-normal scale family (точный g_rr = C/r²), общая теорема для scale families (Appendix F). GAP 1″, GAP 2″ частично закрыты на уровне примеров/расширений. |
| **IDT Theorem M leq kappa Corrected** | PDF | **Теорема M ≤ κ:** M(ω) ≤ κ = λ_max/λ_min с равенством тогда и только тогда, когда спектр изотропен (κ = 1). Доказательство: λᵢ² ≤ λₙ·λᵢ. Следствия: разрыв (κ−M)/κ как сигнатура IDT (5–25% в 3D); таблица предсказаний (абсолютный τ, τ_ratio, зависимость от T, gap Δ). Исправления по сравнению с v2.0: условие равенства, элементарная оценка вместо Коши–Шварца, явная положительность λᵢ. |

## 2. Сводка: что в каких документах доказано или заявлено

- **Теоремы (доказаны в тексте):**  
  - Structural Inertia = время релаксации (IDT-SI, Theorem 1).  
  - M(ω) ≤ κ, равенство iff изотропия (Theorem M ≤ κ).  
  - Уникальность d = 3 для устойчивого связанного режима при Φ_eff класса A/r² − B/r^(d−2) (IDT-D3).  
  - Dimensional Structural Inertia Law m*(d) = C r*² / [2A(4−d)] (вывод из IDT-SI + IDT-D3).  
  - g_rr = C/r² для регулярных scale families (Math Foundations, D3 Appendix F).

- **Численные/экспериментальные результаты (описаны в документах):**  
  - **NN01:** Experiment B (ошибка < 1e-12), Experiment C (32× изменение τ при фиксированном H), эксперименты 1–5, G-inversion 5/5 сидов (τ ratio 1.21×), перестановочный аудит 7 сидов (4/7 C1, 2/7 C2, 89.8% pctile — AMBIGUOUS). Все это для ODE и MLP/Digits, **не** для ResNet-20/CIFAR-10.  
  - **ResNet-20/CIFAR-10:** в idt gpu tests.pdf и README описан **протокол** и код; **таблиц или отчётов о прогонах (pctile_M, tau_ratio, FULL PASS/FAIL) в репозитории нет**. NN01 явно называет этот эксперимент «решающим открытым» и отсылает к GPU-коду в Appendix B.

- **Заявленные следствия без эмпирической проверки в репозитории:**  
  - IITD, Mass-Gap, Differential Inertial Spectroscopy, Geometric Inversion Protocol, IDT-Predictor, IDLM Q-Factor (GAP3 Plan) — протоколы и предсказания, без приведённых данных измерений.  
  - Связь m*(3) с физической массой (Dimensional Inertia Law) — помечена как conjecture, зависит от GAP-3.

## 3. Обновление аудита с учётом всех документов

### 3.1 Что меняется относительно предыдущего аудита

- **Научный вклад:** Теоретическая часть не «только рекомбинация»: есть аксиоматика (A0–A5), канонические выводы (Z, Φ, G, Ляпунов, устойчивость), теоремы (Structural Inertia = τ_relax, M ≤ κ, размерный отбор d=3, Dimensional Inertia Law). Рабочая статья NN01 добавляет **описанные** численные эксперименты (ODE, MLP) с отчётными таблицами и вердиктом по перестановочному аудиту. Таким образом, **новизна и строгость выше**, чем если бы учитывались только GPU-тесты.  
- **Результаты тестов:** Результаты **есть** для ODE и MLP/Digits (NN01); **нет** для ResNet-20/CIFAR-10 — в репозитории ни разу не приведены итоги запуска idt_gpu_tests.py (ни в PDF, ни в README, ни в отдельном отчёте).  
- **Связь теория ↔ тесты:** Спецификация GPU-тестов (Test A, B) прямо соответствует описанному в NN01 «следующему шагу» (ResNet-20/CIFAR-10, bs=32 vs 512, τ = итерации до ΔL=0.02). Формула M(ω) и критерии (C1–C4, tau_ratio, sign_acc) согласованы с NN01 и GAP3.

### 3.2 Что остаётся в силе

- **ResNet-20/CIFAR-10:** Полноценной валидации по **этим** тестам по-прежнему **нет**: нет ни одного отчёта о прогоне с метриками и вердиктом.  
- **Один датасет и одна архитектура** для GPU-сюита; нет сравнения с другими методами из литературы, абляций и явной статистики (ДИ, мощность) в документах.  
- **Ценность для LLM/AI:** по-прежнему не прослеживается в материалах (фокус — CNN, оптимизация, информация-геометрия).  
- **Бизнес-ценность:** в документах не обоснована.

## 4. Итоговое заключение по полному набору документов

- **Теория:** В репозитории представлена связная аксиоматическая и теоретическая база (IDT): аксиомы A0–A5, канонические выводы, теоремы о структурной инерции, M ≤ κ, размерном отборе и законе размерной инерции. Статусы выводов (canon-derived / extension / conjecture) в документах указаны.  
- **Численная проверка (до GPU):** В NN01 приведены результаты по ODE (B, C, 1–5) и по нейросетям (MLP/Digits, G-inversion 5/5, перестановочный аудит с вердиктом AMBIGUOUS). Это **уже** не «только протокол без результатов».  
- **GPU-тесты (ResNet-20/CIFAR-10):** Спецификация и код совпадают с «следующим шагом» из NN01, но **ни в одном документе репозитория не приведены результаты запуска** (нет final_report.json, нет таблицы по сидам). Поэтому для **именно этого** сценария вывод предыдущего аудита сохраняется: полноценной валидации нет.  
- **Рекомендация:** Для полной доказательной базы по IDT в нейросетях необходимо: (1) запустить idt_gpu_tests.py (например, --test AB --seeds 10), (2) включить в репозиторий или в отдельный отчёт таблицу метрик и вердикт (FULL/PARTIAL/AMBIGUOUS/FAIL), (3) при желании усилить выводы — добавить несколько архитектур/датасетов и явную статистику (ДИ, мощность).

Данный файл дополняет **RESEARCH_AUDIT_REPORT_RU.md**: там оценка сфокусирована на тестовых PDF и коде; здесь учтены **все** просмотренные теоретические и рабочие документы репозитория.
